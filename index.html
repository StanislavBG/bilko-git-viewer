Privacy Policy · last updated 2026-06-14

Bilko Git Viewer is a personal app that displays a static snapshot of the developer's own public GitHub activity statistics.

Data we collect

None. The app does not collect, store, transmit, or share any personal data. It contains no accounts, no login, no analytics, no advertising, and no tracking of any kind.

Network & storage

All statistics shown are bundled inside the app as a static file at build time. The app does not make network requests to fetch your data and does not write any personal data to your device.

Third parties

No data is shared with third parties because no data is collected.

Children

The app is safe for all ages and collects no data from anyone, including children.
